## create new branch
1. 建立新的分支
```bash
git checkout -b 你的分支名稱
```
2. 確認是不是切到你的分支
```bash
git branch
```
他會列出你正在工作的分支(前面有星星那個就是你正在工作的分支)
![](https://i.imgur.com/AkHujOn.png)
